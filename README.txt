Documentation du Script de Traitement de Phrase

Ce script Python permet de saisir une phrase, d'analyser son contenu, et d'afficher diverses informations à son sujet.

Utilisation :
1. Exécutez le script Python.
2. Saisissez une phrase lorsque vous y êtes invité.

Fonctionnalités :
1. Nombre de mots : Le script compte le nombre de mots dans la phrase.
2. Nombre de caractères : Il compte également le nombre total de caractères dans la phrase, y compris les espaces.
3. Longueur moyenne des mots : Il calcule la longueur moyenne des mots dans la phrase.
4. Mot le plus long : Le script identifie le mot le plus long dans la phrase.
5. Mots en ordre alphabétique : Il affiche les mots de la phrase dans l'ordre alphabétique.

Exemple d'utilisation :
Entrez une phrase : Python est un langage de programmation puissant et polyvalent.

Résultat attendu :
Nombre de mots dans la phrase : 9
Nombre de caractères dans la phrase : 49
Longueur moyenne des mots : 5.44 caractères
Mot le plus long : programmation
Mots dans l'ordre alphabétique : Python, est, de, langage, programmation, puissant, et, un, polyvalent.
